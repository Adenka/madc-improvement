# MADC Improvement

This script is designed to compare the achievable and converse bounds between results presented in paper https://arxiv.org/pdf/2206.12851 with our own results.

This script generates a diagram that visually represents the comparison of these bounds.
By default, the script runs for parameters `Lambda = 15`, `r = 2` and iterates over `alpha`,
however you are welcome to change these parameters in the script to suit your needs.
The diagram generated by this script with the default parameters is present in our paper.

To run the script, execute the following commands (providing that you have Python 3 installed):
```
pip install -r requirements.txt
python3 bounds-compare-diagram.py 
```
